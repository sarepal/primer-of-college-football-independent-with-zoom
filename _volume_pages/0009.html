{% raw %}PREFACE{% endraw %}
{% raw %}The fact that several very excellent{% endraw %}
{% raw %}books have been written upon football{% endraw %}
{% raw %}during the past few years would seem{% endraw %}
{% raw %}to leave little or no room for another.{% endraw %}
{% raw %}The excuse for adding another may{% endraw %}
{% raw %}be found in the wish to provide a{% endraw %}
{% raw %}primary work for the great army of{% endraw %}
{% raw %}beginners within and outside of the{% endraw %}
{% raw %}Fitting Schools.{% endraw %}
{% raw %}If the writer has succeeded, even in{% endraw %}
{% raw %}a small way, in teaching the youthful{% endraw %}
{% raw %}football idea how to shoot, so to say,{% endraw %}
{% raw %}he will have been amply repaid for his{% endraw %}
{% raw %}pains.{% endraw %}
