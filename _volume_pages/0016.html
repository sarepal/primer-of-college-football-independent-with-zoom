{% raw %}2 A PRIMER OF COLLEGE FOOTBALL{% endraw %}
{% raw %}Football is more like military science{% endraw %}
{% raw %}than any other. It has been well called{% endraw %}
{% raw %}"a game of war." The training of the{% endraw %}
{% raw %}individual is not unlike that of the sol¬{% endraw %}
{% raw %}dier. The player must understand the{% endraw %}
{% raw %}fundamentals of catching, passing, drop¬{% endraw %}
{% raw %}ping, etc., the same as the soldier does{% endraw %}
{% raw %}his manual of arms. He must learn{% endraw %}
{% raw %}position playing—end, tackle, half-back,{% endraw %}
{% raw %}etc.—the same as the soldier that of pri¬{% endraw %}
{% raw %}vate, sergeant, captain. The whole body{% endraw %}
{% raw %}considered, the eleven must know it.?{% endraw %}
{% raw %}plays, direct and strategic, just as the{% endraw %}
{% raw %}company or battalion its marches and{% endraw %}
{% raw %}counter-marches. Offensive and defen¬{% endraw %}
{% raw %}sive tactics are equally necessary to both.{% endraw %}
{% raw %}Superb generalship is just as requisite for{% endraw %}
{% raw %}the former as the latter. The rigid disci¬{% endraw %}
{% raw %}pline of the army is the discipline of the{% endraw %}
{% raw %}eleven.{% endraw %}
{% raw %}Football has this in common with all{% endraw %}
{% raw %}other sciences: that only so much can be{% endraw %}
{% raw %}learned from the books and blackboard.{% endraw %}
