{% raw %}8 A PRIMER OF COLLEGE FOOTBALL{% endraw %}
{% raw %}The body should be well forward and{% endraw %}
{% raw %}knees bent, similar to the position of a{% endraw %}
{% raw %}bowler. See Fig. 2.{% endraw %}
{% raw %}The overhand pass is of dubious value,{% endraw %}
{% raw %}though used by some quarter-backs. One{% endraw %}
{% raw %}thing may be said : It never goes over the{% endraw %}
{% raw %}head of the half-back. If missed, it can{% endraw %}
{% raw %}be easily fallen upon. The pass starts{% endraw %}
{% raw %}from taking of the snap, the arm being{% endraw %}
{% raw %}carried above the shoulder, going through{% endraw %}
{% raw %}about a three-quarter circle and then{% endraw %}
{% raw %}going off on a tangent.{% endraw %}
{% raw %}The toss is generally made in getting{% endraw %}
{% raw %}the ball to the back when plunges are{% endraw %}
{% raw %}made through the flanks of the line. This{% endraw %}
{% raw %}should be done with both hands, as it is{% endraw %}
{% raw %}safer and just as quick, though, as often,{% endraw %}
{% raw %}it may be done with one hand. The toss{% endraw %}
{% raw %}must be made quickly and before the{% endraw %}
{% raw %}quarter moves from his position.{% endraw %}
{% raw %}Handing the ball is used in giving the{% endraw %}
{% raw %}ball to the back trying to advance through{% endraw %}
{% raw %}the centre. The ball should be picked up{% endraw %}
