{% raw %}A Dead Ball. — Of course the ball is{% endraw %}
{% raw %}very seldom dead in a game, but it may{% endraw %}
{% raw %}sometime occur that twenty-two men are{% endraw %}
{% raw %}after a ball that has nearly lost all motion.{% endraw %}
{% raw %}In that case, the most skilful man at div¬{% endraw %}
{% raw %}ing upon the ball will be more than likely{% endraw %}
{% raw %}to get it, because a man can reach a ball{% endraw %}
{% raw %}when in diving distance quicker than he{% endraw %}
{% raw %}could run the same distance. The player{% endraw %}
{% raw %}should run to within about one and a half{% endraw %}
{% raw %}times his length from the ball and then{% endraw %}
{% raw %}leave tho ground, the same as a swimmer{% endraw %}
{% raw %}making a dive, care being taken to land{% endraw %}
{% raw %}upon the ball, to offset the force of gravity.{% endraw %}
