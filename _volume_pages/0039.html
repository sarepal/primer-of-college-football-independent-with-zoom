{% raw %}FUNDAMENTALS{% endraw %}
{% raw %}15{% endraw %}
{% raw %}The requirements for good kicking are{% endraw %}
{% raw %}cool-headedness, a good eye, a good leg,{% endraw %}
{% raw %}and a good square-toe shoe (or nearly so),{% endraw %}
{% raw %}reinforced at bottom.{% endraw %}
{% raw %}There are three kinds of kicks : a Punt,{% endraw %}
{% raw %}a Place Kick,, and a Drop. Of these, the{% endraw %}
{% raw %}most important is the punt. A punt, as{% endraw %}
{% raw %}defined by the rules, is a kick "made by{% endraw %}
{% raw %}letting the ball fall from the hands and{% endraw %}
{% raw %}kicking it before it touches the ground."{% endraw %}
{% raw %}There are two kinds of punts — a com¬{% endraw %}
{% raw %}mon punt, which is generally used, and{% endraw %}
{% raw %}a "twister," or "floater." The latter,{% endraw %}
{% raw %}which is of not much importance, is made{% endraw %}
{% raw %}by dropping the ball so that the longer{% endraw %}
{% raw %}axis will be horizontal, or at right angles,{% endraw %}
{% raw %}to the body (ends pointing to and away{% endraw %}
{% raw %}from the body), and giving the impetus to{% endraw %}
{% raw %}the ball a little to one side. The twist is{% endraw %}
{% raw %}given, says a famous player, " by cutting{% endraw %}
{% raw %}as a tennis ball is cut with a racket."{% endraw %}
