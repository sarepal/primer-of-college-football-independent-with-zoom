{% raw %}5. Blocking.—Good blocking is one of{% endraw %}
{% raw %}the primary essentials of the offensive{% endraw %}
{% raw %}game. No play can be started unless the{% endraw %}

{% raw %}Position of the Body in Blocking.—The{% endraw %}
{% raw %}player should get his body into a posi¬{% endraw %}
{% raw %}tion which is mechanically the strongest,{% endraw %}
{% raw %}his build and playing position considered.{% endraw %}
{% raw %}That is to say, the strongest position for{% endraw %}
{% raw %}a short man would not be the strongest{% endraw %}
{% raw %}position for a tall man, and a guard must{% endraw %}
{% raw %}never stand as high as an end. Generally{% endraw %}
{% raw %}the position of the body which is strong¬{% endraw %}
{% raw %}est is the angular form. Many players{% endraw %}
{% raw %}stand with the trunk at right angles with{% endraw %}
{% raw %}their leg, but it is decidedly weak.{% endraw %}
