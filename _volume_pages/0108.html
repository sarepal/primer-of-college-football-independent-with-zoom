{% raw %}60 A PRIMER OF COLLEGE FOOTBALL{% endraw %}
{% raw %}In offensive work a guard has very few{% endraw %}
{% raw %}assignments, except when he runs with the{% endraw %}
{% raw %}ball or heads a heavy interference; but{% endraw %}
{% raw %}his value is not measured by the number{% endraw %}
{% raw %}of plays in which he may be specially as¬{% endraw %}
{% raw %}signed. As a general rule, he must block{% endraw %}
{% raw %}low and long.{% endraw %}
{% raw %}In blocking, a guard must get low—how{% endraw %}
{% raw %}low will depend largely upon the build of{% endraw %}
{% raw %}the man. About the correct height of a{% endraw %}
{% raw %}six-footer is when the middle finger of his{% endraw %}
{% raw %}right hand can touch the ground. He{% endraw %}
{% raw %}should bend his neck in order to see the{% endraw %}
{% raw %}man in front of him, and not sit back on{% endraw %}
{% raw %}his haunches to do so. A short man built{% endraw %}
{% raw %}close to the ground may stand higher, but{% endraw %}
{% raw %}he is in danger of being carried off his{% endraw %}
{% raw %}feet at any moment by the man playing{% endraw %}
{% raw %}under him.{% endraw %}

{% raw %}next to centre forward if possible; but if{% endraw %}
