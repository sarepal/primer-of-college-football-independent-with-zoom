{% raw %}82 A PRIMER OF COLLEGE FOOTBALL{% endraw %}
{% raw %}5. The Half-Back.—It is exceedingly{% endraw %}
{% raw %}difficult to write upon this position, es¬{% endraw %}
{% raw %}pecially for one who has never played it.{% endraw %}
{% raw %}But next to actual experience are study{% endraw %}
{% raw %}and observation for a number of years,{% endraw %}
{% raw %}and upon these the writer relies. Much{% endraw %}
{% raw %}general drool might be easily written, but{% endraw %}
{% raw %}to write something that would be helpful{% endraw %}
{% raw %}to the beginner is a different matter.{% endraw %}
{% raw %}The positions of half back and full¬{% endraw %}
{% raw %}back have generally been treated as one{% endraw %}
{% raw %}by most writers. Indeed, there are not{% endraw %}
{% raw %}wanting those who believe that the backs{% endraw %}
{% raw %}should be interchangeable, that each one{% endraw %}
{% raw %}of the three should be able to play in{% endraw %}
{% raw %}either position. It seems to the writer{% endraw %}
{% raw %}that this is a great mistake. The high¬{% endraw %}
{% raw %}est efficiency is attainable only along the{% endraw %}
{% raw %}lines of specialism. Hence, after selec¬{% endraw %}
{% raw %}tion, the left half should play at left half,{% endraw %}
{% raw %}the right half at right half, and the full¬{% endraw %}
{% raw %}back at full-back.{% endraw %}
