{% raw %}POSITION PLAY{% endraw %}
{% raw %}93{% endraw %}
{% raw %}The full-back should be chosen almost{% endraw %}
{% raw %}solely for his ability to kick. Other qual¬{% endraw %}
{% raw %}ifications are desirable, to be sure, but the{% endraw %}
{% raw %}ability to kick is the prime requisite. He{% endraw %}
{% raw %}should be a well built, strong - legged,{% endraw %}
{% raw %}active, and speedy fellow, perfectly cool{% endraw %}
{% raw %}and fearless. Weight should be about the{% endraw %}
{% raw %}same as that of half-back. The prelim¬{% endraw %}
{% raw %}inary training of the full-back should be{% endraw %}
{% raw %}one long continuous kick. By constant{% endraw %}
{% raw %}practice only can he acquire skill in this{% endraw %}
{% raw %}branch of the game. At the same time,{% endraw %}
{% raw %}other fundamentals must not be neg¬{% endraw %}
{% raw %}lected. Catching will come with the{% endraw %}
{% raw %}kicking.{% endraw %}
{% raw %}Tackling is perhaps the next in impor¬{% endraw %}
{% raw %}tance.{% endraw %}
{% raw %}The full-back must ofttimes kick in the{% endraw %}
{% raw %}midst of the opposing rushers, and it takes{% endraw %}
{% raw %}a cool head, steady nerve, and strong leg{% endraw %}
{% raw %}to get the ball away. He must catch a{% endraw %}
{% raw %}punt with the opposing rushers all around{% endraw %}
{% raw %}him, when to muff means a touch-down{% endraw %}
