{% raw %}rules allow three men to start before the{% endraw %}

{% raw %}The three fastest men should be selected{% endraw %}
{% raw %}for the flying start, preferably the two{% endraw %}
{% raw %}ends and a back. The ends should be{% endraw %}
{% raw %}out in the wings of the line, and the back{% endraw %}
{% raw %}near the centre; either one of the backs-{% endraw %}
{% raw %}full-back, if he be not the kicker—should{% endraw %}
{% raw %}be back at about the forty-yard line, to{% endraw %}
{% raw %}look out for a return. The other players{% endraw %}
{% raw %}should be lined up on either side in equal{% endraw %}
{% raw %}numbers, and at intervals far enough apart{% endraw %}
{% raw %}to sweep the field. See Diagram 5.{% endraw %}
{% raw %}The ball should be placed and kicked{% endraw %}
{% raw %}by the best place-kicker on the eleven, it{% endraw %}
{% raw %}matters not which position he plays.{% endraw %}
{% raw %}There should be a signal for the direction{% endraw %}
{% raw %}of the kick. When the captain says{% endraw %}
{% raw %}"Ready," the players at the centre line{% endraw %}
{% raw %}should get on their mark, and get away{% endraw %}
{% raw %}with the booting of the ball, or a little be¬{% endraw %}
