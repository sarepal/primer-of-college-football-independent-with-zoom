{% raw %}132 A PRIMER OF COLLEGE FOOTBALL{% endraw %}
{% raw %}There is a very good criss-cross between{% endraw %}
{% raw %}tackle and end. The end should be near{% endraw %}
{% raw %}the side line, say over on the right. Let{% endraw %}
{% raw %}the left tackle run twice, and on second{% endraw %}
{% raw %}run give it to the end who has the long{% endraw %}
{% raw %}field, and if speedy he should make a good{% endraw %}
{% raw %}run.{% endraw %}
{% raw %}Massing of the men on one side of the{% endraw %}
{% raw %}line, so as to draw the opponents over, and{% endraw %}
{% raw %}then attacking the other side, is another{% endraw %}
{% raw %}trick, as in right end and tackle over.{% endraw %}
{% raw %}There are many formations, plays, or tricks{% endraw %}
{% raw %}from queer line-ups. There are the two{% endraw %}
{% raw %}general classes of tricks: tricks from or¬{% endraw %}
{% raw %}dinary line-up, and tricks from formations.{% endraw %}
{% raw %}But there are also plays in which the{% endraw %}
{% raw %}ball is concealed, as the famous play used{% endraw %}
{% raw %}by Stagg's team, in which the opponents{% endraw %}
{% raw %}were drawn out towards flank and a run¬{% endraw %}
{% raw %}ner sent through the centre.{% endraw %}
{% raw %}There are also a number of plays de¬{% endraw %}
{% raw %}vised by Mr. Deland, in which right-angled{% endraw %}
{% raw %}blocking is used.{% endraw %}
