{% raw %}The quick kick and the bluff kick are{% endraw %}
{% raw %}also in the same category—i.e.,belong to{% endraw %}
{% raw %}the strategic game.{% endraw %}
{% raw %}A new trick used last season for the{% endraw %}
{% raw %}first time is the scrimmage kick. The{% endraw %}
{% raw %}centre puts the ball down as if to snap it{% endraw %}
{% raw %}back (the rule allows him to put the ball{% endraw %}
{% raw %}into play by either kicking it forward or{% endraw %}
{% raw %}snapping it back). Instead of snapping it{% endraw %}
{% raw %}back, he kicks between the opponent's{% endraw %}
{% raw %}legs. The object is to kick about ten{% endraw %}
{% raw %}yards, although that is not necessary, since{% endraw %}
{% raw %}the rule as to distance of kicks refers to{% endraw %}
{% raw %}start-off, or when perhaps a kick is called{% endraw %}
{% raw %}for. If the ball strikes an opponent and{% endraw %}
{% raw %}the same side that kicked it gets the ball,{% endraw %}
{% raw %}it is a first down, having been put on side{% endraw %}
{% raw %}by touching an opponent.{% endraw %}
