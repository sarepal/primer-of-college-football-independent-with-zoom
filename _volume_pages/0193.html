{% raw %}(d) Signals.—A Signal is a sign of some{% endraw %}
{% raw %}kind given to indicate to'player the play{% endraw %}
{% raw %}to be used and the time of its execution.{% endraw %}
{% raw %}There are two kinds of signals, the{% endraw %}
{% raw %}vocal signal and the visible Bignal—the one{% endraw %}
{% raw %}addressed to the ear and the other to the{% endraw %}
{% raw %}eye-{% endraw %}
{% raw %}Signals should be as simple as possible,{% endraw %}
{% raw %}so as to be easily understood by the side{% endraw %}
{% raw %}using them. The tendency is to make{% endraw %}
{% raw %}them rather complicated, in order to make{% endraw %}
{% raw %}them unintelligible to the other side; and{% endraw %}
{% raw %}the captain framing his code of signals{% endraw %}

{% raw %}score. The result is that the signals are{% endraw %}
{% raw %}more mystifying to his own men than to{% endraw %}
{% raw %}the other side. Avoid by all means a{% endraw %}
{% raw %}complex system.{% endraw %}
