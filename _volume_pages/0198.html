{% raw %}(e) Generalship.—The subject of offen¬{% endraw %}
{% raw %}sive Generalship is one that is difficult to{% endraw %}
{% raw %}treat in a satisfactory way. Every one{% endraw %}
{% raw %}knows what it is, but to express it is an¬{% endraw %}
{% raw %}other thing.{% endraw %}
{% raw %}It is not the petty politics or jockeying{% endraw %}
{% raw %}before a game, but it is the conduct of{% endraw %}
{% raw %}the game in actual progress upon the{% endraw %}
{% raw %}field. Good generalship consists simply{% endraw %}
{% raw %}in doing the right thing at the right time{% endraw %}
{% raw %}and place. But every act must be upon{% endraw %}
{% raw %}some special line or plan of offence. The{% endraw %}
{% raw %}one object of the side on the offensive to{% endraw %}
{% raw %}which it bends its every energy is to score,{% endraw %}
{% raw %}to carry the ball across the opponents'{% endraw %}
{% raw %}goal line. Good generalship consists in{% endraw %}
{% raw %}taking it there, and its only criterion is{% endraw %}
{% raw %}success, notwithstanding the fact that the{% endraw %}
{% raw %}defeated team may show as good general¬{% endraw %}
{% raw %}ship as the successful one.{% endraw %}
