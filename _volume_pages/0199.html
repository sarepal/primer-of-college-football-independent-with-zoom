{% raw %}THE OFFENCE{% endraw %}
{% raw %}147{% endraw %}
{% raw %}The generalship of the game devolves{% endraw %}
{% raw %}upon the captain, and rightly so. There{% endraw %}
{% raw %}must be one head on the field, and only{% endraw %}
{% raw %}one. There is no time for consultation{% endraw %}
{% raw %}upon the field. That is the place for{% endraw %}
{% raw %}action. The captain must command, the{% endraw %}
{% raw %}others obey.{% endraw %}
{% raw %}A game may be largely planned before{% endraw %}
{% raw %}going upon the field. When the game is{% endraw %}
{% raw %}being mapped out is the time for consul¬{% endraw %}
{% raw %}tation with coaches and players. Before{% endraw %}
{% raw %}the game, it may be decided what is to be{% endraw %}
{% raw %}done under given conditions of wind and{% endraw %}
{% raw %}weather, or what is to be done if the team{% endraw %}
{% raw %}gets the ball at start-off or not.{% endraw %}
{% raw %}By studying the opponents from their{% endraw %}
{% raw %}preceding games, it is possible to deter¬{% endraw %}
{% raw %}mine somewhat in advance the kind of{% endraw %}
{% raw %}game that is likely to succeed against that{% endraw %}
{% raw %}particular team.{% endraw %}
{% raw %}The strength and weakness of the team{% endraw %}
{% raw %}must be considered also —what are its{% endraw %}
{% raw %}strongest parts and its weakest points.{% endraw %}
