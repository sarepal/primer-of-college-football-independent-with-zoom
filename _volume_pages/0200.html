{% raw %}148 A PRIMER OF COLLEGE FOOTBALL{% endraw %}
{% raw %}First, consider the matter of general¬{% endraw %}
{% raw %}ship without reference to the opposing{% endraw %}
{% raw %}team. There are two ways of advancing{% endraw %}
{% raw %}the ball: one, by kicking; the other, by{% endraw %}
{% raw %}rushing. The rushing game is divided{% endraw %}
{% raw %}into straight football and strategic.{% endraw %}
{% raw %}There are practically three schools of{% endraw %}
{% raw %}football: the simple straight football,{% endraw %}
{% raw %}the strategic, and the kicking. The right{% endraw %}
{% raw %}use of these different methods of advanc¬{% endraw %}
{% raw %}ing the ball, the proper proportion of each{% endraw %}
{% raw %}kind of plays, are the great problems of{% endraw %}
{% raw %}good generalship.{% endraw %}
{% raw %}Simple straight football should form{% endraw %}
{% raw %}the basis of the offensive game. It is{% endraw %}
{% raw %}more easily executed, and is less exhaust¬{% endraw %}
{% raw %}ing upon body and mind. A trick re¬{% endraw %}
{% raw %}quires the doing of so many things by{% endraw %}
{% raw %}each individual at a given time that there{% endraw %}
{% raw %}is produced a great mental strain. Men{% endraw %}
{% raw %}begin to worry and wonder whether the{% endraw %}
{% raw %}trick will succeed. And if a fine trick{% endraw %}
{% raw %}fails, they despair of the success of any-{% endraw %}
