{% raw %}Enthusiasm, and not excitement, is what{% endraw %}
{% raw %}is wanted. Too rapid a succession of plays{% endraw %}
{% raw %}results in a jumble merely, and a sort of{% endraw %}
{% raw %}feverish excitement, instead of deadly ex¬{% endraw %}
{% raw %}ecution. Still it must not be understood{% endraw %}
{% raw %}that a calm, deliberate sort of a game is{% endraw %}
{% raw %}the one to be played. When the oppos¬{% endraw %}
{% raw %}ing team is on the run, there should be no{% endraw %}
{% raw %}let-up in the fire. As the advancing party{% endraw %}
{% raw %}gets nearer the goal, the harder, faster,{% endraw %}
{% raw %}and more aggressive should be the game.{% endraw %}
{% raw %}No time should be given the other side{% endraw %}
{% raw %}to pull itself together, but it should be{% endraw %}
{% raw %}driven back and over the goal line; then{% endraw %}
{% raw %}it is time to rest.{% endraw %}
