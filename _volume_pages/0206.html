{% raw %}THE DEFENCE{% endraw %}
{% raw %}The general understanding is that the{% endraw %}
{% raw %}side not having the ball is on the defen¬{% endraw %}
{% raw %}sive, but in reality a team is on the defen¬{% endraw %}
{% raw %}sive as long as it is in its own territory,{% endraw %}
{% raw %}whether it has the ball or not. Defence{% endraw %}
{% raw %}will be used here in the ordinary sense:{% endraw %}
{% raw %}the side not having the ball.{% endraw %}
{% raw %}The importance of a systematic and{% endraw %}
{% raw %}scientific defence is emphasized by the{% endraw %}
{% raw %}fact that it is just one-half of the game.{% endraw %}
{% raw %}A team with a perfect defence, while it{% endraw %}
{% raw %}may never win, certainly can never lose.{% endraw %}
{% raw %}Ori the other hand, it may be able to score{% endraw %}
{% raw %}itself, but if it cannot keep the other side{% endraw %}
{% raw %}from scoring, may lose in spite of a fairly{% endraw %}
{% raw %}good offensive game.{% endraw %}
