{% raw %}THE DEFENCE{% endraw %}
{% raw %}159{% endraw %}
{% raw %}off is the following: Place the tackles,{% endraw %}
{% raw %}centre, and guards on the forty-five-yard{% endraw %}
{% raw %}line or ten yards from centre. Their duty{% endraw %}
{% raw %}is to block the speediest forwards on the{% endraw %}
{% raw %}other side; they can at least delay them{% endraw %}
{% raw %}by forcing them to go outside of them.{% endraw %}
{% raw %}The reason for placing them on that line{% endraw %}
{% raw %}is that the nearer you are to a man the{% endraw %}
{% raw %}better you can block him. The field is{% endraw %}
{% raw %}broken quicker, and gives the backs a{% endraw %}
{% raw %}fair chance for either a return run or{% endraw %}
{% raw %}kick.{% endraw %}
{% raw %}The ends are placed on the thirty-yard{% endraw %}
{% raw %}line. They are to look out for kicks be¬{% endraw %}
{% raw %}tween them and the first line. The quar¬{% endraw %}
{% raw %}ter stands mid-field on the twenty-yard{% endraw %}
{% raw %}line to cover boundary between ends and{% endraw %}
{% raw %}backs.{% endraw %}

{% raw %}and the full-back on the five-yard line to{% endraw %}
{% raw %}cover remainder of territory. The scheme,{% endraw %}
{% raw %}in a word, is that the six fastest men cover{% endraw %}
