{% raw %}THE DEFENCE{% endraw %}
{% raw %}177{% endraw %}
{% raw %}best way to defend one's own territory is{% endraw %}
{% raw %}to advance into the other fellow's.{% endraw %}
{% raw %}The same rule applies when a team is{% endraw %}
{% raw %}pushed back into either corner of its terri¬{% endraw %}
{% raw %}tory. In both cases, the end on the long{% endraw %}
{% raw %}side of the field should keep a sharp look¬{% endraw %}
{% raw %}out for a dash around the end.{% endraw %}
{% raw %}When a team gets possession of the ball{% endraw %}
{% raw %}in close proximity to its own goal line, it{% endraw %}
{% raw %}is really on the defensive, as has been said{% endraw %}
{% raw %}elsewhere. The ball must be booted from{% endraw %}
{% raw %}behind its own goal line, more than likely.{% endraw %}
{% raw %}Then it is that the full-back should exer¬{% endraw %}
{% raw %}cise great judgment; and if the opposing{% endraw %}
{% raw %}forwards are through the line and on him{% endraw %}
{% raw %}before he can get in his kick, he should{% endraw %}
{% raw %}allow a safety, and not give the other side{% endraw %}
{% raw %}a touch-down. Better give up two points{% endraw %}
{% raw %}than four or six.{% endraw %}
